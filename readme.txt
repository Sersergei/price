База данных: 
	MySQL

Среда разработки: 
	PHP, MySQL, HTML, Javascript (jquery)

Требования:
	Страницы необходимо разработать с использованием ООП и MVC.
	
Приветствуется:
	Использование своей верстки предоставленных страниц в HTML5
	Креативный подход к задаче

Описание задания: 
	Необходимо разработать веб-приложение, состоящее из одной веб-страницы 
	(products.html) на которой размещен список товаров и форма добавления
	нового товара.
	При клике на изображение delete.gif, товар удаляется
	(удаляется запись в БД и загруженное изображение) и список обновляется.
	При добавлении нового товара все поля формы обязательны к заполнению!
	При клике на кнопку Add добавляется новый товар с проверкой данных формы,
	после чего форма очищается и новый товар появляется в списке.
	При клике на цену ранее добавленного товара, цена стает редактируемая и
	возле цены появляются кнопки Сохранить и Отменить. При нажатии на сохранить
	цена обновляется на сервере с помощью AJAX запроса).

Хотелось бы, чтобы Вы продемонстрировали:
	понимание ООП и MVC,
	читабельность кода,
	оптимизацию базы данных (индексация полей и т.п.),
	умение работать с JavaScript и AJAX
